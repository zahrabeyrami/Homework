# تمرین‌های درس ساختمان داده
سلام استاد، این پروژه شامل کدهایی هست که در طول ترم سر کلاس گفتین و چیزهایی هست که یاد گرفتم .

### مواردی که در این فایل کد زدم:
* **صف حلقوی:** که با استفاده از باقی‌مانده (درصد) نوشتیم تا فضای لیست هدر نره.
* **پشته (Stack):** همون ساختار LIFO که با دستورات append و pop پایتون کار می‌کنه.
* **لیست پیوندی:** ساختن گره‌ها (Node) و وصل کردنشون به همدیگه.
* **درخت و گراف:** پیاده‌سازی‌های اولیه برای درخت جستجوی دودویی و گراف ساده.

### چطوری کد رو ببینید؟
همه توابع و کلاس‌ها داخل فایل `data_structures.py` هستن.
